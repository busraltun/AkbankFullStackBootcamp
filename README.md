# Akbank Full Stack Bootcamp

Bu repo içerisinde Akbank ve Kodluyoruz' un organize ettiği Full Stack Bootcamp' i çerçevesinde yapılan çalışmaları verilen görevler neticesinde tamamlanan projeleri, çalışmaları içermektedir. Bu bootcamp içerisinde ;

* .Net Core API Giriş & Http Kavramı & Model - Validations & Routing & API Call

* OOP & Pattern Yaklaşımı & EFCore & 3rd party open source .NET libraries

* SQL Server & N-Tier Architecture & Repository Pattern Implementation With Dapper & CleanCode
* Authentication & ExceptionHandling & Caching
* BackgroundWorker & Windows Service
* JavaScript
* Intro to React
* React Cont.
gibi konu başlıkları içermekte. Ayrıca konularla alakalı bahsi geçmeyen içeriklere de değinildi.
